\begin{abstract}
\noindent This report presents the data-classification phase of the Seeding
QDArchive project. Forty independently collected student databases were
aggregated into a single deduplicated corpus of 89{,}658 qualitative-research
projects, harmonised against a shared schema. Each project was assigned a project
type (QDA, QD, other, or non-project) derived from its file inventory, and
classified against the ISIC Rev.~5 taxonomy at the division level using both
file-level and metadata signals, with free-form tags generated to support search
and discovery. The results are reported per source repository, covering
project-type composition, primary-class distributions, and ranked class
frequencies.
\end{abstract}
